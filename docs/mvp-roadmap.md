# ViraFlow MVP Roadmap

## Positioning

ViraFlow is a creator-commerce social app with one clear promise:

Create + Go Viral + Make Money

## Phase 1

- Authentication flow
- Language selection
- Bottom-tab mobile shell
- Reels feed with like, comment, repost
- Follow and unfollow creators
- Marketplace listing flow
- Profile management
- Subscription UI
- AI tools placeholder

## Phase 2

- Real PostgreSQL or Supabase persistence
- File and video storage
- Stripe checkout
- Notifications
- Better marketplace seller contact flow

## Phase 3

- AI image generation
- AI caption generator
- AI script and hook generator

## Phase 4

- AI video generation integrations
- Templates
- Dubbing
- Avatars

## Phase 5

- Referral loops
- Creator monetization analytics
- Sponsored reels
- Web admin dashboard
